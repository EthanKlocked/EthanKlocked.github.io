* root 계정 혹은 sudo 권한 계정으로 진행

### 1. 권한관리

* 계정생성
``` bash
sudo useradd -m newuser

-- 옵션 -- 
-m : 해당 유저의 폴더를 같이 생성
-g : 그룹 지정
-d : 디렉토리 지정
-s : 쉘(shell) 지정
-p : 패스워드(암호) 지정
  
```
