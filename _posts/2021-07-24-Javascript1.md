# # JavaScript 언어의 구동원리
일반적으로 실행되는 코드의 구동원리부터 살펴보면, 

1. 코드를 작성 (프로그래밍 언어)
2. 작성된 코드를 컴파일 혹은 인터프리팅 등의 과정을 통해 기계어로 변환 (0과 1)
3. 변환된 기계어를 메모리에 저장
4. CPU가 메모리에 저장된 기계어 코드를 실행
